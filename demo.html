How it works

  • Each face is a hinge pivoting on a shared edge; one progress 0→1 drives every fold angle to flat.
  • Render at 128² (anti-aliasing) → drawImage downscale to 32² → toDataURL('image/png') → swap <link rel="icon">.
  • Plays once, then freezes the final frame. Favicon updates capped at ~30 fps.
Reduced-motion is on — showing the unfurled frame statically, no animation.

Notes & browser support

Where the animation runs
  • Firefox — animates the tab icon smoothly, even backgrounded.
  • Chrome / Edge — animates while the tab is focused; throttles to ~1 fps when backgrounded.
  • Safari — the real tab icon may not update; the mock tab strip above always shows it.
Why these choices
  • preserveDrawingBuffer:true so the WebGL canvas can be read without coming out blank.
  • One-shot-on-load + freeze sidesteps background throttling and CPU drain.
  • Honors prefers-reduced-motion; falls back to a static inline-SVG cube.
